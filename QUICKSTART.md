## Installation

### Download

1. Visit the [Releases page](https://github.com/yourusername/MindfulDesk/releases)

1. Run the `.msi` installer
2. Follow the installation wizard
3. Launch MindfulDesk from Start Menu

#### macOS
1. Open the `.dmg` file
2. Drag MindfulDesk to Applications folder
3. Launch from Applications

```bash
sudo dpkg -i mindful-desk_0.1.0_amd64.deb

```

#### Linux (AppImage)
```bash
chmod +x MindfulDesk_0.1.0_x64.AppImage
./MindfulDesk_0.1.0_x64.AppImage
```

## Troubleshooting

### Timer Won't Start

1. Check if another session is already running
2. Try stopping and restarting the session
3. Restart the application

### Notifications Not Working

1. Go to **Settings** tab
2. Ensure "Enable notifications" is checked
3. Check your OS notification settings
4. Grant notification permissions to MindfulDesk

### Dark Theme Not Switching

1. Click the theme toggle button
2. Ensure JavaScript is enabled
3. Try restarting the application

### Data Not Saving

1. Check write permissions for the app data folder
2. Verify disk space is available
3. Check if database file exists (see Privacy section above)
